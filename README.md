logtacts
========

Better contact management.

To get started:
 - `git clone git@github.com:phildini/logtacts.git`
 - Set up a virtualenv
 - `pip install -r requirements.txt`
 - `python manage.py runserver`

To run tests:
 - `python manage.py test contacts`
